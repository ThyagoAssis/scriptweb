Comandos :
    Criação do projeto:  ng new nome_projeto
    Criação de componentes: ng generate component nome_componente
    Criação de módulos: ng generate module nome_modulo
    Criação de interfaces: ng generate interface nome_interface
    Colocar o projeto no ar: ng serve --open

Instalação do Bootstrap no projeto: 
    Passo 01: Acessar a pasta do nosso projeto ANGULAR: 
    Passo 02: Executar pelo o seguinte comando:  npm install bootstrap --save
    Passo 03: Dentro do arquivo angular.json, 
              Adicionar a seguinte linha dentro da opção style:
              "./node_modules/bootstrap/dist/css/bootstrap.min.css"
    Passo 04: Reiniciar o servidor Angular
